The Penny-Wise Algorithm

When multiple users try to subscribe simultaneously, standard billing systems block or get confused because they can't link the single "Rs. 100" bank alert to a specific buyer.

PennyWise elegantly resolves this constraint:

  • When User A starts checkout, they are assigned the exact base price of Rs. 100.00. This amount is reserved in the database for 10 minutes.
  • When User B starts checkout 1 minute later, the system sees that Rs. 100.00 is active. It allocates Rs. 100.01 to User B.
  • When User C starts checkout, they are assigned Rs. 100.02.
  • When a customer transfers money, the bank sends an SMS to your phone. The system parses the SMS, extracts the exact amount (e.g. 100.01) and completes the matching transaction for User B instantly.

Connect Your Physical Phone's SMS Inbox

Since your bank credit alerts are sent to your personal registered phone number, you can automatically forward these SMS alerts to this live application. This entirely bypasses the need for costly enterprise APIs! Follow this 2-minute setup guide.

1
Download MacroDroid on your Android Phone
Search for "MacroDroid - Device Automation" on the Google Play Store and install it. It's a free, lightweight, and incredibly reliable automation tool.
2
Create a New Macro & Set the Trigger
Click Add Macro in MacroDroid.
  • Click + on the Red Trigger panel.
  • Go to Device Events → SMS Received.
  • Select SMS from Any Number (or choose specific numbers like SBI-Alert, HDFCBank, etc.).
  • Set the text match criteria to: Contains.
  • Enter keywords (comma separated) like: credited, received, deposited.
3
Set the Action (Make HTTP Request)
Now add the forwarding endpoint details:
  • Click + on the Blue Actions panel.
  • Go to Applications → HTTP Request.
  • Set the Method to POST.
  • Enter the following Webhook URL (replace with your server's public domain or Local IP):
http://:3000/api/sms
  • Set Content Type to application/json.
  • In the **Request Body**, paste the following JSON payload exactly:
{ "sender": "{sms_number}", "body": "{sms_message}", "timestamp": "{system_time}" }

*MacroDroid will automatically replace {sms_number} and {sms_message} with the actual bank's phone number and the text of the transaction alert!

4
Save and Turn It On!
Give your macro a name (e.g., "Forward Payments to PennyWise") and save it. That's it! Every time your bank sends you a credit notification SMS, your phone will instantly post it to your server, which parses the cents and unlocks the subscription in milliseconds!
