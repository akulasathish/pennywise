*   When a user visits your site, Cloud Run instantly boots up a micro-container to serve the request.
*   When there is no traffic, Cloud Run **scales down to exactly 0 active instances**.
*   **If 0 instances are running, you are billed absolutely $0.00.** You only pay for the exact milliseconds your code is actively processing an incoming request.
